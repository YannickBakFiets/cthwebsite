Github cheatsheet website van YannickBakFiets

  1. Eerst moet je een github repository aanmaken op github waar jij je bestand wilt uploaden. Dan open je CMD of Terminal en dan ga je jezelf locaten in de map waar je wilt zijn. Je gebruikt cd.. om terug te gaan naar de vorige map en cd (naammap) om naar die map toe te gaan
  2. Als je nog geen map hebt: - Type in windows search bar: CMD - In de terminal type je mkdir dan maak je een folder aan in die directory - Dan type je explorer om te kijken je hem kan openen.
  3. Als je bij de folder hebt aangemaakt: - type in explorer navigation bar cmd om de directory te openen in cmd/terminal.
  4. - Andere optie: type cmd in windows search
  5. - Ga naar de map door cd naammap te gebruiken en cd.. om terug te gaan
  6. - git init dit maakt een .git folder aan in de map
  7. - git add (naambestand)
  8. - git commit
  9. - m “first commit”
  10. - git branch
  11. - M main
  12. - git remote add origin https://github.com/YannickBakFiets/test.git
  13. - git push
  14. - u origin main
